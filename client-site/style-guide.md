# Eggs 101 - Style Guide

## Colors

- **Background:** #FFFFFF (white)
- **Card Backgrounds:** #F5F5F5 (light gray)
- **Buttons / Headings:** #FFD700 (vibrant yellow)
- **Secondary Accent / Hover States:** #FFC107 (darker yellow)
- **Body Text:** #333333 (dark gray)

## Fonts (Google Fonts)

- **Headings:** [Fredoka One](https://fonts.google.com/specimen/Fredoka) - rounded, playful, fun
- **Body:** [Nunito](https://fonts.google.com/specimen/Nunito) - clean, readable, slightly rounded

## Design Elements

- **Navbar:** Logo centered, sticking out from the bottom of the nav bar. One page link on each side.
- **Buttons:** Full-width, yellow background with darker yellow on hover
- **Cards:** Lighter gray background with subtle border for all the sections/cards.
- **Joke banners:** Dashed border, light yellow background, italic text, centered
- **Images:** Full-width within sections, cover-fit with consistent height (300px for sections, 220px for cards)
- **Sections:** Cards, stacked vertically with small gaps between them
- **Footer:** Centered, minimal (name, class, year)
- **Tone:** Clean and modern layout, rounded borders, vibrant egg themed colors
