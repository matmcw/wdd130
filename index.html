WWR

White Water Rafting

This project was the first one completed for WWD 130. It was a great learning project that allowed me to practice my HTML and CSS skills. I created a website for a fictional white water rafting company, showcasing their services and activities.

The website features a hero section with a background image, a grid layout for the activities, and a contact card. I used various HTML elements and learned about stlying and fonts.
